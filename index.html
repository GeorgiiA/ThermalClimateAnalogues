Thermal Climate Analogues

A repository of layers for geovisulization of climate model outputs using MapWindow 4

View on GitHub Download .zip Download .tar.gz

Welcome to Thermal Climate Analogues repository.

This is an initial stage of the project aimed to provide geovisualization of climate model outputs in the form of the past and future analogues for the present local climates. The references to the sources of information can be found in the paper available at http://www.nature.com/articles/srep24784. Detailed description of the files contained in the repository is to be added soon.

Добро пожаловать в репозитарий термических аналогов климатических регионов

Этот проект находится сейчас на начальной стадии. Его целью является геовизуализация результатов моделирования прошлых и будущих состояний климатической системы в виде карт термических аналогов для ряда современных климатических регионов.
